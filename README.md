# RangeCalender
Library to show Calender which takes from Date and To date. Useful while generating reports,book tickets etc.

Steps to integrate:

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  dependencies {
	        implementation 'com.github.PalashDumbare:RangeCalender:Tag'
	}
